# Data-Intensive Programming - Assignment

## General info
Simple Spark course assignment for calculating different values from the data located in the data folder.

## Technologies
Created with:
* Python 3.10
* PySpark 3.3.0
* NumPy 1.23.4
* Matplotlib 3.6.2
